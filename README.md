# Power BI Projects Portfolio

This repository contains **interactive Power BI dashboards** built to demonstrate data visualization, analysis, and business intelligence skills. Each project showcases the use of **Power BI Desktop**, **DAX measures**, **trend analysis**, and **interactive visuals** such as **cards, line charts, bar charts, tree maps, donut charts, maps, and slicers**.

## Key Skills Demonstrated
- Data modeling and transformation
- Creating measures and calculated columns using DAX
- Trend analysis and moving averages
- Interactive visualizations with slicers and filters
- Page navigation and dashboard design
- Geographical and product segmentation analysis
- Dashboard layout and storytelling for business insights

## How to Use
1. Download the `.pbix` file from this repository.
2. Open it in **Power BI Desktop** (latest version recommended).
3. Optional: Modify visuals or add your own data to customize the analysis.

## Notes
- Datasets may be sourced from public platforms like Kaggle.
- Sensitive or large raw datasets are not included; refer to project description for source links.
- All dashboards are designed for professional portfolio use and CV showcase.
